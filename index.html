Critical GoAnywhere MFT Flaw: Immediate Patch for Command Injection Risk

2025-09-20

Fortra has issued an urgent security alert and subsequent updates to address a maximum severity vulnerability (CVE-2025-10035) in its GoAnywhere Managed File Transfer (MFT) solution. The flaw, located within the License Servlet, is a deserialization vulnerability that could be exploited by remote threat actors through command injection attacks. This means an attacker could potentially execute arbitrary commands on affected systems, posing a significant risk. Users are strongly advised to apply the security patches immediately to protect their systems against potential exploitation and maintain robust cybersecurity posture.
